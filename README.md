# genecache

```
NAME:
   genecache - cli for caching all dictybase genes

USAGE:
   genecache [global options] command [command options] [arguments...]

VERSION:
   1.0.0

COMMANDS:
     help, h  Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --log-level value        log level for the application (default: "info")
   --log-format value       format of the logging out, either of json or text (default: "json")
   --input value, -i value  Input file with list of paired dictybase gene and transcript ids[required]
   --log-file value         Name of the output log file, default goes to STDERR
   --url value, -u value    Base url for dictybase (default: "http://dictybase.org")
   --help, -h               show help
   --version, -v            print the version
```
